# Mini-Projet S2 - Page de Profil Personnel

## Présentation

Ce projet a été réalisé dans le cadre du Mini-Projet S2 sur les bases du HTML5 sémantique.

L'objectif est de créer une page de profil personnel en utilisant les balises HTML5 sémantiques, sans ajouter de CSS personnel. La mise en page est assurée par le fichier `semanstyle.css` fourni.

## Fonctionnalités

* Structure HTML5 sémantique
* En-tête avec un menu de navigation
* Présentation personnelle
* Section des compétences
* Utilisation des balises `article` et `aside`
* Formulaire de contact accessible
* Pied de page

## Structure du projet

```text
.
├── index.html
├── semanstyle.css
└── README.md
```

## Technologies utilisées

* HTML5
* CSS (fichier fourni)

## Lancer le projet

1. Cloner le dépôt ou télécharger les fichiers.
2. Ouvrir le fichier `index.html` dans un navigateur web.
